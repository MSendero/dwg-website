Data Working Group Meeting

Graph Genomes and their applications

Event Details

The meeting will help direct GA4GH API development over the next year and explore the utility of graph representations of the reference genome. In discussing the GA4GH APIs we'll address driving projects, more expansive query languages and how we can best grow the network of GA4GH compliant data sharing services. For graph genomes we'll dive into the results of the recent graph-genomes bakeoff evaluation and discuss the data structures and algorithms we'll need as we move to employing genome scale graphs in our collective analyses.
